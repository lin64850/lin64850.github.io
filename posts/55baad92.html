步骤1. 单击“开始”并键入“命令提示符”。

步骤2. 然后，右键单击“命令提示符”并选择“以管理员身份运行”。

步骤3. 之后，输入“diskpart”并按“回车”。然后依次执行以下命令：

  • list disk
  • select disk n
  • clean
  • create partition primary
  • format fs=fat32 quick（如果要格式化为NTFS，请将fat32替换为ntfs）
